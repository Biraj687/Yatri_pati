# ✅ Pre-Launch Checklist - YatriPati Header

Use this checklist to ensure everything is ready before launching your YatriPati news portal header.

## 🔧 Installation & Setup

- [ ] Node.js 18+ installed
- [ ] `npm install` completed successfully
- [ ] No dependency conflicts
- [ ] `node_modules/` contains all packages
- [ ] React 19 and react-icons installed

## 🚀 Development Server

- [ ] `npm run dev` starts without errors
- [ ] Server runs on `http://localhost:5173`
- [ ] Hot reload working (changes reflect instantly)
- [ ] No console errors on page load
- [ ] No TypeScript compilation errors

## 🎨 Components Rendering

- [ ] TopBar displays correctly
  - [ ] Nepali date shows (left side)
  - [ ] Breaking news visible on desktop
  - [ ] Social icons visible (4 icons)
  - [ ] Icons are responsive to viewport

- [ ] Navbar displays correctly
  - [ ] Logo with airplane icon visible
  - [ ] Navigation menu visible on desktop
  - [ ] Search icon visible
  - [ ] Dark mode toggle visible
  - [ ] Hamburger menu visible on mobile

- [ ] Header combines both correctly
  - [ ] TopBar above Navbar
  - [ ] Proper spacing between components
  - [ ] No overlapping elements

## 📱 Responsive Design

- [ ] Mobile (< 640px)
  - [ ] Navigation hidden, hamburger menu visible
  - [ ] Breaking news hidden
  - [ ] Date visible but smaller text
  - [ ] Social icons fit properly
  - [ ] No horizontal scrolling

- [ ] Tablet (768px - 1024px)
  - [ ] Layout appears balanced
  - [ ] Navigation partially visible or in menu
  - [ ] Breaking news may be truncated
  - [ ] All icons accessible

- [ ] Desktop (>= 1024px)
  - [ ] Full layout visible
  - [ ] Navigation menu displayed horizontally
  - [ ] Proper spacing throughout
  - [ ] All features accessible

## 🌓 Dark Mode

- [ ] Dark mode toggle button works
- [ ] Clicking toggle switches themes
- [ ] All components update colors:
  - [ ] Background changes
  - [ ] Text color changes
  - [ ] Border colors update
  - [ ] Icon colors update
- [ ] Sun/moon icon changes appropriately
- [ ] No color contrast issues in dark mode

## 🎯 Styling

- [ ] Tailwind CSS applied correctly
- [ ] All colors appear as designed
- [ ] Spacing looks consistent
- [ ] Fonts render properly (Noto Sans Devanagari)
- [ ] No color bleeding between elements
- [ ] Borders display correctly
